About this Project

Tech Stack & Model: This project is built with HTML, CSS, and vanilla JavaScript on the front-end. The prediction engine uses OpenAI's GPT for analyzing market questions and a Random Forest machine learning model (an ensemble of decision trees) to compute the final "Yes" probability:contentReference[oaicite:4]{index=4}. Data is sourced from the public APIs of Manifold Markets.

Disclaimer: This is an unofficial, personal project created for fun and educational purposes. Predictions are not guaranteed to be accurate and should not be taken as financial advice. The site is not affiliated with Manifold Markets or any official trading platform. Use it at your own risk!

How GPT is Integrated: The system uses GPT to interpret each market question's context and nuances. Those AI-generated insights are combined with historical data and fed into the Random Forest model, which then outputs a predicted probability for a "YES" outcome. In short, GPT helps understand the question, and the Random Forest uses that understanding (plus data) to make a prediction.
